# Contributor Covenant Code of Conduct

## 私たちの誓約

私たちはメンバー、貢献者、リーダーとして、年齢、体の大きさ、目に見える・見えない障害、
民族性、性的特徴、性自認と性表現、経験の度合い、学歴、社会経済的地位、国籍、外見、
人種、カースト、肌の色、宗教、性的指向にかかわらず、ハラスメントのない環境での
コミュニティへの参加を可能にすることを誓います。

オープンで、友好的で、多様性があり、インクルーシブで、健全なコミュニティを醸成する
行動を約束します。

## 私たちの基準

コミュニティの前向きな雰囲気に貢献する行動の例:

- 他の人々に対する共感と優しさを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを行い、また受け入れる
- 過ちから学び、影響を受けた人々に謝り、経験から学ぶ責任を持つ
- 個人にとってだけでなく、コミュニティ全体にとって最善なものに注目する

容認されない行動の例:

- 性的な言葉、イメージの使用、およびあらゆる種類の性的な注目や誘い
- 荒らし、侮辱的・中傷的なコメント、個人攻撃、政治攻撃
- 公私を問わないハラスメント
- 住所や電子メールなどの他者の個人情報を、明示的な許可なく公開すること
- 職業上の場において、合理的に不適切だとみなされるその他の行為

## 施行責任

コミュニティリーダーは、受け入れ可能な行動の基準を明確にし、施行する責任を負います。
不適切、脅迫的、攻撃的、または有害と思われる行動に対し、適切かつ公正な是正措置を
取ります。

## 適用範囲

この行動規範は、プロジェクトのすべての空間に適用されます。また、個人が公式に
コミュニティを代表しているときは、公の場にも適用されます。

## 施行

虐待的、ハラスメント的、その他容認されない行動は、[SECURITY.md](./SECURITY.md) に
記載された経路で報告できます。すべての苦情は速やかに公正に審議されます。

## 施行ガイドライン

コミュニティリーダーは、この行動規範に違反すると判断した行動に対し、以下の
コミュニティへの影響ガイドラインに沿って対応します。

1. **修正** — 非公開の書面による警告、公開の謝罪が要求される場合あり
2. **警告** — 一定期間の交流制限
3. **一時的な BAN** — コミュニティからの一時的な排除
4. **永久的な BAN** — コミュニティからの永久排除
